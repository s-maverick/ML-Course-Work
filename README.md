# Machine Learning Course Work

This repository contains a comprehensive collection of machine learning implementations, ranging from foundational statistical models to advanced deep learning architectures. It focuses on the end-to-end ML lifecycle, including exploratory data analysis (EDA), feature engineering, model selection, hyperparameter tuning, and performance evaluation using rigorous validation techniques.

### Tech Stack & Tools

* **Languages:** Python (Scikit-learn, XGBoost, LightGBM).
* **Deep Learning:** PyTorch, TensorFlow, Keras.
* **Data Manipulation:** NumPy, Pandas, SciPy.
* **Visualization:** Matplotlib, Seaborn, Plotly.
* **Experiment Tracking & Tools:** Kaggle Notebooks and Google Colab, MLflow, Optuna (Tuning).
